Quick Start

Search Videos 
GET /api/search?query=python+tutorial&limit=10

Response:
{
  "success": true,
  "query": "python tutorial",
  "count": 10,
  "results": [...]
}
Get Video Details 
GET /api/video/dQw4w9WgXcQ

Response:
{
  "success": true,
  "video_id": "dQw4w9WgXcQ",
  "title": "...",
  "author": "...",
  "view_count": "..."
}
Batch Requests 
POST /api/batch

Body:
[
  {"endpoint": "search", "params": {"query": "music"}},
  {"endpoint": "trending", "params": {}}
]
